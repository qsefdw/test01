# 1. 제목 (Headers)
`#`의 개수에 따라 제목의 크기가 결정됩니다. (1개부터 6개까지 지원)

# 가장 큰 제목 (h1)
## 중간 제목 (h2)
### 소제목 (h3)
#### 더 작은 소제목 (h4)

---

# 2. 텍스트 강조 (Emphasis)
글씨에 스타일을 적용하여 강조할 수 있습니다.

* **굵게 표시하기**: 글자 위아래를 `**` 또는 `__`로 감쌉니다. (예: **중요한 내용**)
* *기울여 표시하기*: 글자 위아래를 `*` 또는 `_`로 감쌉니다. (예: *이탈릭체*)
* ~~취소선 넣기~~: 글자 위아래를 `~~`로 감쌉니다. (예: ~~삭제된 내용~~)

---

# 3. 목록 (Lists)
순서가 있는 목록과 없는 목록을 지원합니다.

### 순서가 없는 목록 (별표, 대시, 플러스 모두 가능)
* 사과
- 바나나
+ 오렌지
    * 탭(Tab)키나 공백을 넣으면 하위 목록이 됩니다.

### 순서가 있는 목록
1. 첫 번째 할 일
2. 두 번째 할 일
3. 세 번째 할 일

---

# 4. 링크와 이미지 (Links & Images)
대괄호`[]`와 소괄호`()`를 조합하여 사용합니다.

* **링크**: [네이버 바로가기](https://www.naver.com) `[표시할 글자](URL 주소)`
* **이미지**: ![로고](https://via.placeholder.com/150) `![이미지 설명](이미지 주소)` (앞에 느낌표 `!`가 붙음)

---

# 5. 코드와 코드 블록 (Code)
프로그래밍 코드나 서식 없는 텍스트를 표현할 때 사용합니다. 키보드의 숫자 1 왼쪽에 있는 백틱(`) 기호를 사용합니다.

* **인라인 코드**: 문장 중에 `이렇게` 짧은 코드를 넣을 때는 백틱 1개로 감쌉니다.
* **블록 코드**: 여러 줄의 코드를 넣을 때는 위아래를 백틱 3개(```)로 감쌉니다. 
  시작하는 백틱 3개 뒤에 언어 이름(c, python, javascript 등)을 적으면 문법 강조(Highlighting)가 됩니다.

```c
#include <stdio.h>
int main() {
    printf("Hello, Markdown!\n");
    return 0;
}
